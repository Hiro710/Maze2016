# 自動迷路探索アプリ

1. Windows環境での作動を推奨しています(Mac非推奨)

2. Windowsの スタートメニュー > アクセサリー > コマンドプロンプト を開き、 cd コマンドで任意の保存場所へ行き、コマンド'$bcc32 maze20161010.c'を実行し、
   その後'$maze20161010' を実行すると迷路探索アプリが起動します
